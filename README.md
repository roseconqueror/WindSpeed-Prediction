# 风速时间序列预测 - 机器学习课程项目

## 项目简介
基于多高度气象传感器数据，实现风速的单步与多步预测，对比线性回归、LSTM、Transformer、CNN1D四类模型性能，支持Optuna贝叶斯超参数调优。

## 预测任务
1. **单步预测**：基于历史数据预测下一时刻风速
2. **多步预测A**：8小时历史窗口 → 预测未来1小时风速
3. **多步预测B**：8小时历史窗口 → 预测未来16小时风速
